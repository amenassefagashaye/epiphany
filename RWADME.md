# Assefa Digital Bingo - Backend Server

Real-time multiplayer bingo game server built with Deno.

## Features

- WebSocket-based real-time communication
- Room management (create/join/leave)
- Multiple game types (75-ball, 90-ball, 30-ball, pattern)
- Real-time number calling
- Win verification and prize calculation
- Chat system
- Player management

### Local Development
